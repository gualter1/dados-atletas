#  Projeto devstart trilha 1 dados-atletas

O projeto cria uma classe com os dados dos atletas, nessa classe é criado um constructor onde ele prepara os dados que serão usados nos metodos abaixo.

* calculaCategoria()
* calculaIMC()
* calculaMediaValida()
* obtemNomeAtleta()
* obtemIdadeAtleta()
* obtemPesoAtleta()
* obtemAltura()
* obtemNotasAtleta()
* obtemCategoria()
* obtemIMC()
* obtemMediaValida()


Ao final do projeto é devolvido uma string contendo todos esses dados em formato de texto.
